# Gest2A3Eco

App de escritorio en **Python (Tkinter)** para generar `suenlace.dat` de A3ECO (bancos y facturas).

## Ejecutar
```bash
pip install -r requirements.txt
python main.py
```

## Compilar (Windows)
```bash
pyinstaller --name=Gest2A3Eco --icon=icono.ico --onefile --windowed --add-data "logo.png;." main.py
```
